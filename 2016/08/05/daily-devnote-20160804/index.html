2016.08.04 오늘의 개발 노트

2016년 8월 6일

Spring Boot와 Spring-Loaded 버전에 따른 호환성 문제

Spring Boot를 이용해 개발을 진행 하던 중, 테스트를 위해 Eclipse가 아닌 콘솔에서 실행시킨 적이 있다.
그런데 Eclipse에서는 잘 동작하던 spring-loaded 모듈에서 아래와 같은 오류가 나는 것이었다.

구글링을 하다보니 Spring-Loaded의 GitHub 저장소에 Issue가 올라와 있었고, 쭉 읽어보던 중 버전 문제일 수 있겠다는 것을 알게됐다.

내가 기존 쓰던 버전은 1.2.0.RELEASE였는데, 이것을 최신 버전인 1.2.5.RELEASE로 바꿔줬더니 정상적으로 동작한다.
